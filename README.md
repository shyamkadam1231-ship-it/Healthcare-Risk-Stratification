## Healthcare Risk Stratification Project A complete end-to-end Data Science project that predicts patient risk levels in healthcare settings. ### 🛠️ Tools & Technologies Used - **SQL** — Retrieved and queried patient data from the database - **Excel** — Data exploration and initial analysis - **Python** — Data preprocessing and model building - **Scikit-learn** — Machine Learning model training - **Streamlit** — Interactive web application for predictions ### 📊 Features - Predicts whether a patient is High Risk or Low Risk - Takes inputs: Age, Length of Stay, Treatment Cost, Abnormal Lab Count - Shows Risk Probability score ### 📁 Dataset Patient data including diagnoses, lab results, and outcomes ### 🚀 How to Run 1. Install requirements: `pip install streamlit pandas joblib scikit-learn` 2. Run: `streamlit run Risk_Model.py`
